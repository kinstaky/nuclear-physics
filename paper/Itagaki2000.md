## 理论
K 可以是 1/2 或者 3/2，[[Oertzen2006]] 中提到 K 是价中子的轨道角动量在对称轴（两α的连线）的投影。按我理解，价中子在 p 轨道，氢原子模型中 p 轨道是纺锤形的，$l=1$，磁量子数 $m$ 可以为 0 或者 $\pm1$。$m$ 为 $\pm1$ 时，对应 $\sin\theta$，否则对应 $\cos\theta$。如果我们将对称轴看成 z 轴，那么 $\sin\theta$ 的形状就是垂直于对称轴，也就是 $\pi$ 键；$\cos\theta$ 的性质平行于对称轴，也就是 $\sigma$ 键。我觉得可以认为 $K=m+1/2$，所以 $\pi$ 键对应于 $K=3/2$，$\sigma$ 对应于 $K=1/2$。另一方面，我们可以考虑 $m=0$ 是独特的，而 $m=\pm1$ 是对称的；而从形状来看，平行于对称轴只有一个方向，但垂直于对称轴有两个方向，所以平行于对称轴的 $\sigma$ 对应于 $m=0$，垂直于对称轴的 $\pi$ 对应于 $m=1$。汇总信息如下表

|           | $\pi$        | $\sigma$     |
| --------- | ------------ | ------------ |
| 形状        | 垂直（甜甜圈）      | 平行（纺锤）       |
| 磁量子数 $m$  | 1            | 0            |
| 角动量投影 $K$ | 3/2          | 1/2          |
| 波函数       | $\sin\theta$ | $\cos\theta$ |
这一切都有迹可循了。
